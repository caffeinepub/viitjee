# Specification

## Summary
**Goal:** Add a contact button with email mailto link to enable users to reach out to the VIITJEE team.

**Planned changes:**
- Add a "Contact Us" button to the landing page with a mailto link to viitjeetec@gmail.com
- Position the button in a visible location (header navigation or About/footer section)
- Style the button consistently with the existing warm orange theme

**User-visible outcome:** Users can click the contact button to open their default email client with the VIITJEE email address pre-filled, making it easy to send inquiries.
